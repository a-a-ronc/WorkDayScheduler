# Work Day Scheduler Starter Code
_________________________________

GIVEN I am using a daily planner to create a schedule
WHEN I open the planner
THEN the current day is displayed at the top of the calendar
WHEN I scroll down
THEN I am presented with time blocks for standard business hours of 9am to 5pm
WHEN I view the time blocks for that day
THEN each time block is color-coded to indicate whether it is in the past, present, or future
WHEN I click into a time block
THEN I can enter an event
WHEN I click the save button for that time block
THEN the text for that event is saved in local storage
WHEN I refresh the page
THEN the saved events persist

# Code-Quiz

## Overview

This is a color coded Javascript work day scheduler. The user is able to input an event into a time block within business hours in the past, present or future. 

## Features

- Multiple-choice time blocks on HTML, CSS, and JavaScript.
- Event storage within a certain time frame
- View saved events functionality.
- Usage of local storage
- Usage of JQuery API

## How to Use

1. **State Event**: Click the gray, green or red container to type the event wanted to be stored.
2. **Save Event**: Click the blue save button to save the event into local storage.
3. **Retain Date**: Feel free to refresh the page and still have the event stored in its corresponding time block. 

## Technologies Used

- HTML
- CSS
- JavaScript
- JQuery

## Setup

1. Clone the repository to your local machine.
2. Open the `index.html` file in a web browser.

## Application Structure

- `index.html`: Main HTML file.
- `style.css`: Stylesheet for styling the application.
- `script.js`: JavaScript file containing the application logic.
